WORKFLOW DE JEU
===============

1. engine.startGame('maize_1', 'maize', 1)
   └── Initialise partie en cours

2. engine.useNASAData(['temperature', 'precipitation'])
   └── Enregistre utilisation NASA

3. Joueur configure culture (irrigation, NPK, sol)

4. result = engine.completeLevel(gameData)
   ├── CompetenceSystem.calculateCompetenceScore()

SYSTEME D'ETOILES
=================

0-50 points    → ⭐      (Echec)
50-75 points   → ⭐⭐    (Moyen)
75-100 points  → ⭐⭐⭐  (Excellent)


DEVERROUILLAGE
==============

Niveau suivant debloque si :
  • 1x score >= 75 points (3 etoiles)
    OU
  • 3x scores >= 50 points consecutifs (2+ etoiles)

DONNEES NASA
============

Types disponibles :
  • temperature   → Optimiser periodes plantation
  • ndvi          → Sante vegetation
  • precipitation → Planifier irrigation
  • smap          → Humidite du sol

Limite : 3 utilisations par niveau
